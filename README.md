# Responsive Pricing Cards UI

A clean, modern SaaS pricing section built with HTML, CSS, and JavaScript.

## Features

- **3 Pricing Plans**: Basic, Pro (Most Popular), and Enterprise
- **Responsive Design**: Horizontal cards on desktop, stacked on mobile
- **Interactive Elements**: 
  - Hover effects on cards
  - Highlight Pro plan with dark background and badge
  - Smooth transitions
- **Visual Enhancements**: Grid background pattern, checkmark icons for features

## Layout

- **Mobile-first** responsive design
- **Desktop**: 3-column layout (768px and above)
- **Mobile**: Single-column stack layout
- **Pro Plan**: Scaled up and highlighted with "Most Popular" badge

## Technical Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts (DM Serif Display, VT323)
- patterncraft.fun for background pattern
- No external libraries or frameworks used

## Files

- `index.html` - Main HTML structure
- `styles.css` - Responsive styling
- `script.js` - Feature list enhancements with icons
